# Public Legal Pages Bundle

Last updated: 2026-06-01

This folder is the static hosting bundle for the first iOS submission.
Upload these files to a stable HTTPS host before App Store review:

- `privacy.html`
- `account-deletion.html`
- `support.html`
- `legal-pages-manifest.json`

After hosting, set the production URLs in App Store Connect and EAS:

- `EXPO_PUBLIC_PRIVACY_URL`
- `EXPO_PUBLIC_ACCOUNT_DELETION_URL`
- `EXPO_PUBLIC_SUPPORT_URL`

Keep the markdown source files in `docs/legal/` and these public HTML files in sync before submitting.
